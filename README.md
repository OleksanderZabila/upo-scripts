# УПО Scripts

Застосунок для автоматичної обробки файлів пожежних виїздів.

## Можливості

- Об'єднує аркуші по датах (19.05 / 20.05) в один, відсортований за часом
- Видаляє службові колонки (A, H–Q)
- Автоматично підставляє **номер автомобіля** по підрозділу (ТО 10 / 12 / 13 / 15 / Умань 1 / 2)
- Формула `=MINUTE(G-D)` для розрахунку хвилин відбуття
- Зберігає результат у тій же папці з датою у назві

## Запуск

### Готовий exe
Запустити `УПО Scripts.exe` — відкриється вікно з емблемою УПО.

### З Python (потрібні залежності)
```
pip install customtkinter tksvg openpyxl pandas Pillow
python УПО_Scripts.py
```

## Налаштування номерів авто

При першому запуску натисніть **⚙ Змінити** і введіть держномери для кожного підрозділу.  
Зберігаються у `upo_config.json` поруч з програмою.
